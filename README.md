# Norway Traffic Fines Dashboard

A Streamlit dashboard that visualises the development of Norwegian traffic fine levels (scenario 2023–2026).

### Features
- Line charts showing fine evolution
- Table preview of fines
- Fully interactive Streamlit UI

### Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

### Files
- `app.py` — Streamlit application
- `requirements.txt` — dependencies
- `README.md` — project description
